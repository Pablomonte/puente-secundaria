# Puente a Secundaria

**Herramientas para empezar con confianza**

Este sitio te ayuda a organizarte, entender consignas, responder con claridad, estudiar con mas seguridad y llegar a la secundaria con herramientas concretas.

## Materias

- [Matematica](matematica/index.md) — 8 modulos: desde entender consignas hasta tablas y graficos
- [Lengua](lengua/index.md) — 4 modulos: lectura, escritura, resumen y vocabulario
- [Ciencias](ciencias/modulo-1.md) — Leer, registrar y concluir

## Como usar este material

- **Frecuencia sugerida:** 2 veces por semana
- **Duracion:** 20 a 25 minutos por bloque
- No es una prueba, es un espacio para practicar
- Equivocarse aca sirve para aprender

## Antes de empezar

Mira la seccion de [Bienvenida](presentacion/bienvenida.md) para entender que cambia en la secundaria y que herramientas ya tenes. Luego pasa por [Tecnicas de estudio](presentacion/tecnicas-estudio.md) para prepararte mejor.
