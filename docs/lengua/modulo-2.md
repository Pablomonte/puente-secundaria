# Lengua Puente a Secundaria

## Módulo 2. Escritura formal: texto expositivo, carta y revisión

**Propósito**  
Aprender a escribir textos más claros, organizados y adecuados a distintas situaciones, como se espera al comenzar la secundaria.

## Página 79. Escribir para informar con claridad
**Texto**  
En secundaria no solo se escribe para responder preguntas. También se escriben textos para explicar un tema, presentar información y comunicar ideas con claridad.

**Situaciones de escritura**
- explicar un tema de Ciencias
- presentar información en Sociales
- escribir una nota o carta formal
- redactar una conclusión

**Actividad**
Completá:
- Cuando escribo para informar, tengo que pensar en: ________
- Una escritura clara ayuda a que el lector: ________

**Cierre**  
Escribir bien no es escribir difícil. Es escribir para que otro entienda.

## Página 80. El texto expositivo: qué es y cómo se organiza
**Texto**  
Un texto expositivo sirve para explicar un tema de manera ordenada.

**Partes básicas**
- presentación del tema
- desarrollo con información importante
- cierre o conclusión

**Actividad guiada**
Leé un texto expositivo breve y marcá:
- dónde presenta el tema
- dónde desarrolla la información
- dónde cierra

**Práctica**
Ordená estas partes para formar un texto:
- una oración que presenta el tema
- dos oraciones que lo explican
- una oración final de cierre

## Página 81. Escribo un texto expositivo breve
**Pasos**
1. elijo el tema
2. anoto ideas importantes
3. ordeno la información
4. escribo un borrador
5. reviso si se entiende

**Actividad**
Elegí uno de estos temas y escribí un texto breve:
- la atmósfera
- un ecosistema
- una fuente de energía
- un hecho histórico trabajado en clase

**Ayuda**
Tu texto debería incluir:
- una oración inicial que presente el tema
- dos o tres ideas importantes
- una oración final de cierre

**Consigna de revisión**
Revisá si:
- el tema aparece claro desde el inicio
- las ideas están en orden
- el cierre no repite exactamente lo mismo

## Página 82. La carta formal
**Texto**  
La carta formal se usa para pedir, informar, proponer o reclamar de manera respetuosa.

**Partes de la carta formal**
- lugar y fecha
- destinatario
- saludo inicial
- motivo de la carta
- desarrollo
- cierre
- firma

**Actividad**
Leé una carta formal breve e identificá sus partes.

**Práctica**
Completá este esquema:
- Lugar y fecha:
- Destinatario:
- Motivo:
- Pedido o propuesta:
- Despedida:
- Firma:

## Página 83. Escribo una carta formal
**Situación**
En tu escuela quieren mejorar un espacio común, organizar una campaña o resolver una necesidad del curso.

**Consigna**
Escribí una carta formal a una autoridad escolar para:
- proponer una mejora
- pedir materiales
- solicitar una actividad
- plantear una necesidad del curso

**Recordá incluir**
- un saludo adecuado
- explicación clara del motivo
- un pedido o propuesta concreta
- un cierre respetuoso

**Modelo de inicio**
Sra. Directora:  
Nos dirigimos a usted para...

## Página 84. Revisar también es escribir
**Texto**  
Un texto no se termina cuando se escribe el primer borrador. Revisar es una parte importante del trabajo.

**Qué revisar**
- si el texto responde a la consigna
- si las ideas están ordenadas
- si hay repeticiones innecesarias
- si las oraciones se entienden
- si la puntuación ayuda a leer mejor
- si la ortografía necesita correcciones

**Actividad**
Leé un texto breve con errores y marcá:
- una repetición
- una oración poco clara
- un problema de puntuación
- una palabra a corregir

## Página 85. Mejoro mi escritura
**Texto**  
Corregir no es borrar todo. Muchas veces alcanza con:
- sacar una palabra que sobra
- cambiar una palabra por otra más precisa
- agregar una explicación
- ordenar mejor una oración

**Actividad**
Mejorá estas oraciones:
- La atmósfera es importante. Es importante porque sirve.
- Queremos pedir algo para la escuela porque nos parece algo bueno.
- En el recreo hay problemas. Problemas que pasan siempre.

**Práctica**
Volvé a tu texto expositivo o a tu carta y mejorá al menos:
- una oración
- una palabra
- un signo de puntuación
- una parte del cierre
